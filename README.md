# BigInt
BigInt class for basic calculations

Supports following:
windows x64, x86

addition
substraction
multiplication
division, fractions 
fraction downsizing

Supports operators:
boolean, !=, ==
+, +=, ++(), ()++
-, -=, --(), ()--
*, *=
/, /=

# Compile
Just build BigInt.vcxproj (using VisualStudio) and you should be good to go.
