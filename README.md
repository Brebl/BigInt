# BigNum
BigNum class for Arbitrary-precision arithmetics  
  
Supports following:  
linux  
windows x64, x86  
  
addition  
substraction  
multiplication  
division  
fractions  
fraction downsizing  
  
Supports operators:  
boolean, !=, ==  
+, +=, ++(), ()++  
-, -=, --(), ()--  
*, *=  
/, /=  
  
# Compile  
`git clone https://github.com/Brebl/BigNum.git`  
`git submodule update --init`  
`make`  
  
or,    
build BigInt.vcxproj (using VisualStudio) and you`re good to go.
